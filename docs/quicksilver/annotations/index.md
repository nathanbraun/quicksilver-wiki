# All Quicksilver annotations by page number

**All annotations by page number**

This is the complete list of page annotations. It's manually generated, so if
you create a new entry you should add it to this page.

If you would like to create your own annotations[^1], you can add a new file to the
[git repository](https://github.com/nathanbraun/quicksilver-wiki). Many
annotations were authored by Neal Stephenson, but anyone 
is welcome to add constructive annotations. A constructive annotation is one
that supplies accurate and useful information about some aspect of the book.

[^1]:
    The convention for naming page annotations is "Author Last Name:Author
    First Name:Book Name:Page number:brief excerpt (Author)".
    
The annotations refer to the American hardcover edition, [ISBN
0380977427](https://www.amazon.com/Quicksilver-Baroque-Cycle-ISBN-0380977427/dp/B000NQMAK4).
